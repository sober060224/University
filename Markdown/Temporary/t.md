### 【深度故事素材】

1.  **公方代表的担忧**：在公私合营前，荣毅仁曾暗自担心公方代表是“外行”，会把工厂搞得“鸡犬不宁”。他曾听闻有的公方代表让老板交出账册后，厂里搞得一团糟，工人反而喊出“宁可老板回来”的口号。因此，当得知来申新的公方代表吴一帆是无锡申三的老熟人、懂业务时，他感到“很宽慰” 。
2.  **潘汉年的心事**：在公方代表入驻那晚，潘汉年在红棉酒家请客，情绪不高，对荣毅仁说了一番意味深长的话：“高处不胜寒……是人，总会有心病的”。这句话既像是对荣毅仁的告诫，也隐约流露出潘汉年自身的处境，让荣毅仁和盛康年感到“傻了” 。
3.  **毛泽东的比喻**：1955年，毛泽东用京剧《打渔杀家》中萧桂英的故事来教育工商界，说闹革命不能像萧桂英那样“船行半江，还惦记着门没上锁”，舍不得丢坛坛罐罐。荣毅仁听后深受启发 。
4.  **邓小平的嘱托**：在“火锅宴”上，邓小平当场指定由谷牧与荣毅仁直接联系，解决他工作中的困难。此后，荣毅仁办公室一直高悬邓小平亲笔书写的“戒欺室”匾额，以示警醒 。

### 【深度故事素材】

1.  **与鲁迅的渊源**：1913年，魏如在绍兴府中学堂读书时，鲁迅是该校教员兼监学。魏如曾因领导学生罢课抗议校方迫害学生而被开除。这段经历让他早年就接触到反帝反封建的思想。
2.  **魏如的眼泪**：虽然魏如下定决心带头合营，但这并不意味着轻松。当时被誉为“红色资本家”的荣毅仁在谈及家族企业合营时尚且落泪，魏如内心的挣扎可想而知。好友胡厥文的回忆文章证实，随着时间的推移，看到合营后工人效率提高、厂区扩展，他才变得越来越认同。
3.  **中国第一辆汽车的制造者**：在抗战西迁祁阳期间，魏如与支秉渊等人共同研制出了中国历史上真正意义上的第一辆汽车。这辆车从祁阳驶出，翻山越岭开到重庆参加展览，是中国汽车制造史上重要的一页。
4.  **最后的遗嘱**：1966年10月18日，魏如因膀胱癌病逝。遵照他的遗嘱，家人将他的遗体捐献给上海华山医院供医学解剖研究。这位倾尽心血于实业救国的老人，在生命最后时刻选择了彻底奉献。

### 【深度故事素材】

1.  **向“洋货”宣战的决心**：1920年办火柴厂时，刘鸿生为解决火柴头脱落、赤磷边磨损的“卡脖子”难题，高薪聘请归国化学博士、斥巨资购买进口设备，经过半年多试验，终于研发出“宝塔”牌火柴，成为当时市场标准产品，远销南洋。
2.  **重庆“小伙计”的苦笑**：抗战期间应蒋介石邀请到后方办厂，因资金仰赖官僚资本，各厂董事长都由“四大家族”的人担任。刘鸿生曾苦笑说：“我们刘家的所有资产等于白白奉送给他们，我们将变成一个微不足道的小股东，我这个总经理变成了他们的小伙计了。”
3.  **“不把所有鸡蛋放一个篮子”**：刘鸿生推行多元化经营，曾对儿子解释：“我并没有让我所有的鸡蛋都放在一个篮子里。如果一个企业组织亏损了，其余的还可以赚到大量利润。”这正是他历经风险而屹立不倒的秘诀。
4.  **教育子女的“小老板”规矩**：刘鸿生有10儿3女，全部送往国外留学。子女回国后全部进入刘氏企业从底层做起：长子刘念义做查账员、四子刘念智做普通会计员、八子刘念信做技术员。一次儿子与厂长争执，他立即责备：“你在厂里必须遵守厂里的制度，我不许可你在厂里摆小老板架子。”
5.  **捐资助学的智慧**：创办定海公学（今舟山中学）时，他捐出23.25万银元作为基金，但故意留出缺额发动社会投资。学者钟祥财评价：“他完全有能力独资，但留些额度让大家共享社会声誉，这是在‘照亮’社会风气。”

### 【刘鸿生履历亮点】

- **1888年** 出生于上海，祖籍浙江定海。
- **1906年** 上海圣约翰大学辍学（因拒当牧师）。
- **1909年** 入英商开平矿务局任跑街，后成买办。
- **1920年** 创办苏州鸿生火柴公司，开启实业之路。
- **1930年** 组建大中华火柴公司，成为“火柴大王”。
- **1937年** 任中国红十字会总会副会长，投身抗战救护。
- **1949年** 从香港回北京，受到周恩来接见。
- **1950年** 加入中国民主建国会。
- **1954年** 率先申请公私合营，刘氏企业逐步纳入公有制。
- **1956年10月1日** 病逝于上海，享年68岁。

### 【深度故事素材】

1.  **乌蛇传说**：民间传说中，张小泉为解救被两条乌蛇污染的水井，喝下雄黄酒跳入井中，用大锤砸死交缠的乌蛇。回家后受蛇尾形状启发，造出了第一把剪刀。这个传说解释了剪刀形状的由来，也为品牌增添了神话色彩。
2.  **80家真假“张小泉”**：清乾隆年间，因张小泉剪刀成为贡品声名大噪，杭州城假冒“张小泉”字号的剪刀店多达八十余家，被人戏称为“青山映碧湖，小泉满街巷”。
3.  **张母拦轿**：1890年，因仿冒者屡禁不止，张永年的母亲孙氏效仿古人，在杭州城隍山下拦轿告状。知县被这位老太太的执着打动，亲笔写下“永禁冒用”的告示并刻碑立于店门。这块石碑成为中国近代知识产权保护史的珍贵见证。
4.  **周恩来与张小泉**：1944年，周恩来在重庆参观内迁工厂展览会时，曾参观新中工程公司（魏如创办），而张小泉作为民族工业代表，也在抗战西迁中延续生产。虽无直接记录，但周恩来一贯重视手工业保护，后来毛泽东的“一万年也不要搞掉”批示，正是对手工业保护政策的延续。
5.  **两个“张小泉”的世纪之争**：由于历史原因，杭州和上海长期并存两个“张小泉”品牌，均为地方国有企业，屡次对簿公堂。直到2007年后富春控股通过资本运作，先控股杭州张小泉，再收购上海张小泉，2014年终于将两家合二为一，结束了长达数十年的品牌纷争。

### 【张小泉品牌发展脉络】

- **1628年（明崇祯元年）** 张小泉在杭州大井巷创立“张小泉剪刀店”。
- **1663年（清康熙二年）** 正式启用“张小泉”招牌。
- **18世纪（乾隆年间）** 被列为贡品，名声大振。
- **1890年（光绪十六年）** 张母拦轿告状，官府立碑“永禁冒用”。
- **1915年** 获巴拿马万国博览会二等奖，走向世界。
- **1956年** 公私合营，成立杭州张小泉剪刀厂、上海张小泉刀剪商店。
- **2000年** 国企改制，迈出多元化转变关键一步。
- **2006年** 被认定为第一批“中华老字号”；锻制技艺入选国家级非遗。
- **2007-2014年** 富春控股收购整合，杭州、上海“张小泉”合二为一。
- **2021年** 深交所创业板上市，成为“刀剪第一股”。
- **2022年** “拍蒜断刀门”引发品牌口碑危机。
- **2023年** 被商务部列为“附条件通过”中华老字号复核，面临整改。
