# 科目三理论考试重点总结
## 一、 案例分析与判断
### 1. 案例找关键词
| 关键词 | 违规行为 |
| :--- | :--- |
| 核载/实载/承载 | 超员/超载 |
| 路段/速度/限速 | 超速 |
| 某日/昨日/今日 | 疲劳驾驶 |
| 驾驶证/吊销 | 准驾不符 |
| 酒精浓度/含量 | 酒驾/醉驾 |
### 2. 案例三步搞定
1.  **先看3超**：超员、超速、超载。
2.  **再看人**：喝酒没？疲劳驾驶没？
3.  **看其他**：车有隐患，准驾不符等。
### 3. 五星考点（数据类）
*   **驾驶车型**：A2驾驶大型客车（准驾不符）；中型客车载客>19人（超员）。
*   **速度限制**：
    *   特殊路段 > 30km/h（超速）。
    *   客货高速 > 100km/h（超速）。
    *   无中心线：城市30km/h，公路40km/h（城3公4）。
    *   有中心线：城市50km/h，公路70km/h（城5公7）。
    *   高速公路：最高100km/h，最低50km/h。
*   **酒驾标准**：酒精含量 $\ge 20mg/100ml$。

## 二、 交通信号与标志标线
### 1. 交警手势
| 手势特征 | 含义 |
| :--- | :--- |
| 左高右低 | 停止信号 |
| 双手从未放下（平伸） | 直行信号 |
| 单右手向中间摆动 | 变道信号 |
| 单右手下压 | 减速慢行信号 |
| 单左手下压 | 左转弯待转信号 |
| **易错口诀** | 一只手掌朝前，另一只手在下，哪只手在下朝哪儿转（左/右转弯）。 |
### 2. 易混标志与标线
*   **铁路道口**：火车图标代表无人看管，栅栏图标代表有人看管。
*   **事故与落石**：事故易发 $\neq$ 追尾；有石落石，护栏榜山。
*   **颜色含义**：
    *   标志：红色禁令，黑色解除，黄色三角找注意。
    *   标线：黄高白低（黄色分隔对向，白色分隔同向）。
*   **形状与图示**：正脸代表机动车，侧身对应具体车型；会车标志哪边粗哪边先走。
*   **通行标志**：红圈空白禁通行，红圈横杠禁驶入。
*   **其他**：一人逃跑去避难（疏散），要走安全门，只有终点冲红线；没有无线电只有交通广播频率。
### 3. 核心口诀
*   **让行**：会车哪边粗哪边先走。
*   **停车**：三减八停。
*   **标志等级**：标志红高蓝低黄建议。
*   **标线规则**：虚线可跨，实线不可跨。
*   **答题技巧**：问“应注意”通常先全选。

## 三、 速度、让行与通行规则
### 1. 速度与距离
*   **特殊天气（雾天）**：
    *   能见度<200m：车速60，车距100（261）。
    *   能见度<100m：车速40，车距50（145）。
    *   能见度<50m：车速20，尽快驶离（520）。
*   **特殊路段**：3特找30（特殊路段限速30）。
*   **无线/有线**：城3公4 / 城5公7。
### 2. 让行与超车
| 场景 | 规则 |
| :--- | :--- |
| **让车** | 右让左，弯让直，右方先行。 |
| **停车** | 口5站3（路口/站牌50米内禁停），临3（临时停车危险报警闪光灯）。 |
| **让行人** | 行人过马路，优先找停车。 |
| **让校车** | 同向二车道停车，三车道最左侧可减速通过。 |
| **超车** | 左侧超车，特殊路段/车辆不得超。 |
| **会车** | 靠山让临崖，有障让无障。 |
| **高速** | 入左离右（进高速打左灯，出高速打右灯），禁止倒车掉头逆行。 |
| **跟车** | 保持安全距离，夜间使用近光灯。 |
| **通行原则** | 别人违反、别人不让，我依然要让。 |

## 四、 特殊路段与恶劣天气
| 场景 | 操作要点 |
| :--- | :--- |
| **泥泞路** | 易侧滑，前轮侧滑向相反方向打，后轮侧滑向相同方向打；停车观察、驱动轮下铺石。 |
| **涉水** | 制动效果下降，间断轻踏制动（排刹）。 |
| **隧道** | 开启近光灯，禁止使用远光灯，注意明暗适应。 |
| **铁路** | 一停二看三通过，道口内不换挡。 |
| **山区** | 制动上坡推迟，下坡提前；停车上坡左转（打方向），下坡右转。 |
| **环岛** | 进环岛不打灯，出环岛打右转向灯。 |
| **冰雪** | 制动距离延长，利用发动机制动，循车辙行驶。 |
| **雨天** | 易水滑失控和追尾，看不清停车。 |
| **雾天** | 开启雾灯，遵守“261 145 520”限速规则。 |

## 五、 紧急避险与事故处置
### 1. 五星考点与核心原则
*   **爆胎**：胎压过高/过低都会爆胎。处置：**先稳方向再减速**，非必要不急刹。
*   **事故避险原则**：稳方向、抢挂低档、不急转、不急刹。只有事故无法避免时，可紧急制动。
### 2. 具体事故操作
| 事故类型 | 正确操作 |
| :--- | :--- |
| **转向失控** | 转向困难安全停车，果断连续踩松制动踏板（点刹），**禁止紧急制动**。 |
| **碰撞** | 向碰撞一侧修正。车速较高/高速，可先制动。正面碰撞侧面躲，**禁止跳车**。 |
| **侧滑** | 泥泞影响选侧滑。前轮相反，后轮相同。 |
| **制动失效** | 先减档（利用发动机制动），寻找避险车道、刮擦减速。**禁止越二级减档**，禁止首先拉紧驻车制动（手刹）。 |
### 3. 故障处置流程
1.  **打灯**：开启危险报警闪光灯。
2.  **放标**：车后放警告标志（高速公路150米以外，普通公路50-100米）。
3.  **人转移**：人员转移到右侧路肩或安全地带，**禁止留在车内等待**。
4.  **报警**：报警或找救援/拖拽车。
### 4. 火灾处置
*   **三不**：不得用水灭火（电器/油路），不得在车内等待救援，不得开启发动机罩（防止空气进入加剧燃烧，应微开缝隙灭火）。

## 六、 现场急救常识
### 1. 正确急救措施
*   **出血**：先止血。
*   **中毒**：移出毒区。
*   **烧伤**：喷冷水降温。
*   **动脉出血**：近心端止血。
*   **昏迷**：先检查呼吸，搬运前先开放气道。
*   **其他**：保鲜膜覆盖（保护伤口），三角巾固定，可喝淡盐水。
### 2. 禁止行为（全错）
*   随意移动伤员（除非有危险）。
*   原地等待救援（针对中毒等需立即脱离现场的情况）。
*   用细麻绳包扎。
*   使用软担架（颈椎骨折等伤情禁止）。
